Choosing the Default option on investment type means you're using some very simple math to split the loot. Basically, with this option each member pays the same price (group purchase) and receives equal cuts. The corresponding math can be seen in the default calculation section.

The underlying math is very simple. The cut received by -th member is calculated as:

where is number of members. Each member pays the same amount, thus there are no cases of uneven payouts.

Example: If you hit a Fleeca bank and the payout you received is 213 marked bills, then just divide 213 by how many members are in the group. If there are 3 members, each would receive 71 bags since 213 divided by 3 is 71. Simple.
